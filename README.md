# Recipe App Setup

1. Open `config.js` and update:
   - Site title
   - Author name
   - Categories
   - Default image path

2. Add initial recipes in `recipes.json`. Format:
   - name
   - category
   - url (filename in recipes folder)
   - image
   - ingredients (array)
   - directions (array)
   - notes (optional array)

3. Place images in the `images/` folder.

4. Open `index.html` in a browser. Use the Recipe Builder to create new recipes.

5. Generated recipes will be saved as `.html` files in your downloads. Move them to the `recipes/` folder if you want them permanent.
